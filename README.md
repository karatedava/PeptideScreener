# YOLO STAMP

Peptide Screener

## Installation

First, make sure **uv** is installed: https://docs.astral.sh/uv/getting-started/installation/

```bash
git clone https://github.com/karatedava/yolo_stamp.git
cd PeptideScreener
uv sync 
```

## Running the application

```bash
uv run app.py
```

Open the provided link in your browser
